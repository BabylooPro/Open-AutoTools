# AutoLower

## Description

Converts any text entered by the user to lowercase and automatically copies the result to your clipboard.

## Usage

```bash
autolower "Your text here."
```

## Output

```
your text here.
```

## Examples

```bash
# Basic usage
autolower "HELLO WORLD"
# Output: hello world

# Multiple words
autolower "THIS IS A TEST"
# Output: this is a test
```

# AutoCaps

## Description

Converts any text entered by the user to uppercase and automatically copies the result to your clipboard.

## Usage

```bash
autocaps "Your text here."
```

## Output

```
YOUR TEXT HERE.
```

## Examples

```bash
# Basic usage
autocaps "hello world"
# Output: HELLO WORLD

# Multiple words
autocaps "this is a test"
# Output: THIS IS A TEST
```

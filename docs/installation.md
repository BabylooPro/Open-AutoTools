# Installation

## How to install to use directly

To install Open-AutoTools, use the following command in your terminal:

```bash
pip install open-autotools
```

This command installs all the necessary tools to integrate Open-AutoTools into your workflow.

You can also find the package on PyPI at: https://pypi.org/project/Open-AutoTools/

## Verify Installation

After installation, verify that the tools are available:

```bash
autotools --version
```

You should see the version information and development mode status.
